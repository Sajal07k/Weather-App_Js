# 🌦️ Weather App - JavaScript

A clean and responsive Weather App built using **HTML**, **CSS**, and **Vanilla JavaScript**, which fetches real-time weather data from the **OpenWeatherMap API**.

---

## 🔍 Features

- 🔎 Search for any city and get current weather info
- 🌡️ Displays temperature, description, humidity, and wind speed
- 🌥️ Dynamic weather icons based on condition
- ❌ Shows error message if location not found
- 📱 Responsive and mobile-friendly UI

---

## 📦 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

---

## 🚀 How to Use

1. **Clone the repository**
git clone https://github.com/Sajal07k/Weather-App_Js.git
cd Weather-App_Js
2. **Open `index.html` using Live Server**  
(Right-click > Open with Live Server in VS Code)

---

## 🔐 Setup API Key

You’ll need a free API key from [OpenWeatherMap](https://openweathermap.org/api):

1. Create an account and go to the [API Keys page](https://home.openweathermap.org/api_keys)
2. Copy your API key
3. Open `script.js` and replace this line:
``js
const api_key = "YOUR_API_KEY_HERE";
